Overview:
This project is a data analysis study using FIFA dataset.
The goal is to explore the data, perform analysis, and extract meaningful insights using Python in Google Colab.

Objective:
Explore data using statistics and visualization
Identify patterns and insights

Tools Used:
Python 
Pandas
NumPy
Matplotlib / Seaborn
Google Colab

Insights:
-England has the largest number of players, with 1662 players
-The top 10 countries with the highest number of players are:
-England, Germany, Spain, Argentina, France, Brazil, Italy, Colombia, Japan, Netherlands
-Player overall ratings vary across different years
-There is a negative relationship between age and performance (as age increases, overall performance tends to decrease)
-The peak performance age range is approximately between the 30s and early 40s
